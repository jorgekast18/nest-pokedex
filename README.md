# Execute in develop mood

1. Clone the repository
2. Execute
```
yarn install
```

3. Install NestJs
```
npm i -g @nestjs/cli
```

4. Up Database
```
docker-compose up -d
```

5. Populate DB with the seed
```
http://localhost:3000/api/v2/seed
```

## Stack used
* MongoDB
* Nest
